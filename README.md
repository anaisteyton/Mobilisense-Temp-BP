Repository for the manuscript entitled, "Dynamic Temperature Exposure and Ambulatory Blood Pressure in Paris, France: An Activity Space-Based Approach using the MobiliSense Study," 
co-authored by Anaïs Teyton, Sanjeev Bista, Dikshya Maharjan, Giovanna Fancello, Noémie Letellier, Andrea Montanari, Tarik Benmarhnia, Basile Chaix

This includes three sets of R code:
1. The linear mixed models as well as the ten sensitivity analyses;
2. The visualizations, incluing figures and tables to present LMM findings;
3. The nonlinear mixed models, along with figures created to visualize NLMM findings;

For any questions, please contact Anaïs Teyton, the corresponding author and code developer at ateyton@ucsd.edu
